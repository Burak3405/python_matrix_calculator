# Linear Algebra Solver

A simple command-line calculator for matrix math. It is written in Python and uses the NumPy library.

Features
* Choose Size: You can set the size of your matrix (like 2x2 or 3x3).
* Loop: The program keeps running until as long as you want
* Math Operations:
  * Add matrices
  * Multiply matrices
  * Find Determinant
  * Find Transpose
  * Find Inverse

## Technologies Used
* Python
* NumPy

## How to Run
1. Download or clone this project:
   `git clone https://github.com/Burak3405/python_matrix_calculator.git`
2. Open the folder:
   `cd python_matrix_calculator`
3. Install NumPy:
   `pip install numpy`
4. Run the program:
   `python linear_solver.py`
